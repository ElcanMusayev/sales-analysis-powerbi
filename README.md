# Satış Analizi — Power BI Dashboard

## Layihə haqqında
Ərzaq satış məlumatlarının çoxsəhifəli Power BI dashboardu ilə analizi.

## İstifadə olunan alətlər
- Power BI (DAX, KPI, İnteraktiv Vizuallaşdırma, Tooltip)

## Dashboard səhifələri
- Satış İcmalı — ümumi satış, mənfəət, bölgə analizi
- Məhsul Analizi — 250 unikal məhsul, kateqoriya üzrə pay
- Müştəri və İşçi Analizi — 50 müştəri, ən yaxşı menecerlər

## Nəticələr
- Ümumi satış: 2,68M | Ümumi mənfəət: 1,79M
- Qusar regionu ən yüksək satış həcminə sahib oldu (185K)
- Şirniyyat kateqoriyası ən yüksək satış payını təşkil etdi (22,32%)
- Elnur Həsənov ən yüksək satış göstəricili menecer oldu (105K)
- Dondurulmuş Ərik Dilimi ən çox satılan məhsul oldu (26,390)
